# LAV Obrador

Premium artisan bakery storefront. Next.js 16 + Tailwind 4.

**Live:** https://lav-obrador.vercel.app

## Dev

```bash
npm install
npm run dev
```

## Stack

- Next.js 16 (App Router) + TypeScript
- Tailwind 4 (CSS-first)
- Zustand for cart state
- Fraunces + Geist via next/font
- Unsplash for imagery
